- Sintaxe do Javascript
- Variáveis
- Representação dos tipos de dados

#### [01/04] Aula 02

- Manipulação de string
- Operadores aritméticos e lógicos

#### [03/04] Aula 03

- Operadores aritméticos, lógicos e relacionais
